# AP3 — Del EDA al modelo: predicción de diabetes tipo 2

## Objetivo

Construir un modelo básico de clasificación binaria que prediga la presencia de diabetes tipo 2 a partir de variables clínicas y biométricas, evaluar su desempeño y discutir limitaciones técnicas y éticas.

## Dataset

PIMA Indians Diabetes Database (768 instancias × 9 columnas: 8 variables clínicas + variable objetivo binaria `Outcome`).

- Fuente: [Kaggle — PIMA Indians Diabetes Database](https://www.kaggle.com/datasets/uciml/pima-indians-diabetes-database)
- El archivo `diabetes.csv` está incluido en este repositorio.
